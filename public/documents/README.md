# Research Library documents

The Research Library links to files in **`public/Coral Farming/`** (the extracted Coral Farming archive). Documents and readings open from that folder when you click a card.

To add or replace a file, place it in the same path structure under `public/Coral Farming/` and ensure the `link` in `src/data/researchLibrary.ts` matches the path (with a leading `/`).

**Alternative:** Use external URLs (e.g. Google Drive share links). Edit `src/data/researchLibrary.ts` and set `link: 'https://...'` for any document or reading.
